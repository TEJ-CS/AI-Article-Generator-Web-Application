# 📝 AI Article Generator Web Application

## 🚀 Project Overview
This is a full-stack Flask web application that generates high-quality AI-powered articles using the Groq LLM API.  
Users can sign up, log in, generate articles on any topic, save history, and export articles as TXT or PDF.

---

## ✨ Features

- 🔐 User Authentication (Signup/Login/Logout)
- 🧠 AI Article Generation using Groq LLM (LLaMA 3.3)
- 💾 Save articles in SQLite database
- 📜 View article history per user
- 📄 Export articles as TXT or PDF
- 🌐 Auto-open browser on run

---

## 🛠️ Tech Stack

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- Groq API (LLM)
- SQLite Database
- ReportLab (PDF generation)
- HTML Templates (Jinja2)

🔥 How it works
User enters topic
Groq AI generates article
Article saved in database
User can download as TXT or PDF
